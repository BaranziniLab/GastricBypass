# Figure Source Data — Baseline Metabolomic Profiling of Weight Change After RYGB

**Version:** 1.0
**Companion code repository:** https://github.com/baranzinilab/GastricBypass
**Manuscript:** *Baseline metabolomic profile as a potential biomarker for weight change after Roux-en-Y gastric bypass (RYGB) surgery*

This deposit contains the **de-identified, publication-level source data** underlying the figures of the manuscript, together with summary tables for the Framingham Heart Study (FHS) genetic analyses. It is designed so that every quantitative figure panel can be reproduced from the accompanying code repository without access to controlled individual-level data.

---

## What this deposit is (and is not)

- **Is:** the processed, aggregate or pseudonymized data plotted in each figure panel (trajectory means, ROC curves, odds ratios, correlation matrices, score distributions, network node/edge tables, GWAS hit summaries, and cohort summary statistics).
- **Is not:** raw or individual-level metabolomics, genotype, or phenotype data. Those are governed by the data-use agreements of the source studies (LABS-2, the post-RYGB clinical cohort, the Estonian Biobank, and FHS) and are **not** redistributed here.

### Deliberately excluded for privacy

- **`FigS3/ebw_metab_cohort_site.csv`** (LABS-2 per-participant excess-body-weight records with participant `id`, age, sex, race, and surgical site) was **withheld** because it is quasi-identifiable. The figure it supports can be regenerated by approved investigators using the controlled LABS-2 data.
- Raw per-subject metabolite matrices listed in the repository's `figures/_README.txt` manifest are likewise not included.

**Panel-to-figure mapping and a full description of every panel are in the repository file

## How to reproduce a figure

1. Clone the code repository: `git clone https://github.com/baranzinilab/GastricBypass`
2. Place the relevant `data.csv` into the matching `figures/<panel>/` directory (or point the script at this deposit).
3. From the panel directory: `source("make_plot.R")`.

Plotting conventions (themes, sizes, colorblind-friendly palette) are documented in `figures/plotting-style-reference.md`.

## Data provenance and accessions

- **LABS-2** (discovery): Longitudinal Assessment of Bariatric Surgery-2. Individual-level data via the study's data-access process.
- **Post-RYGB clinical cohort** and **Estonian Biobank** (external validation): available under the respective institutions' agreements.
- **Framingham Heart Study** (genetics): dbGaP accession **phs000007** (consent group HMB-IRB-MDS). Component phenotype tables used: pht000091, pht000202, pht002343, pht003099, pht006027. *Final accession/version details pending author confirmation.*

The FHS cohort-construction and GWAS pipeline is documented in the repository at `tables/README.md`.
