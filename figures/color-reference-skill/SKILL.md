# Color Reference

Use this skill when color choice or color interpretation matters for this project.

## When To Use

Use this skill if the task involves:

- matching the existing figure palette
- choosing colors for a new panel in this project
- interpreting group identity from plotted colors
- reconstructing data from figure colors
- deciding which end of a diverging palette is low vs high

## Core Rule

Do not improvise the palette when a project color convention already exists.

For this project, preserve the established semantic mapping from color to cohort, class, or value direction.

## Important Convention

For the project’s correlation diverging palette:

- lower = `#0072B2`
- midpoint = white
- higher = `#D55E00`

This should be treated as the intended semantic direction when documenting or reconstructing correlation heatmaps and related diverging scales for this project.
